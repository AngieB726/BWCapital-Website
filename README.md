# Black & White Capital — Marketing Site

Static one-page marketing site for **Black & White Capital LLC**, a New Jersey
business management and consulting firm.

## Files

- `index.html` — landing page
- `thank-you.html` — post-inquiry confirmation page
- `styles.css` — all styling
- `CNAME` — custom domain configuration for GitHub Pages

## Local preview

Just open `index.html` in any browser — no build step required.

## Hosting

Deployed via **GitHub Pages** from the `main` branch.
Custom domain managed in `CNAME` and DNS records on Squarespace.

## Inquiry form

The contact form posts to **FormSubmit.co**, which forwards every submission
straight to Christine@BWCapital.org. No backend required.
